# Consulta de Endereço por CEP

Aplicação web desenvolvida com **HTML**, **CSS** e **JavaScript** para consultar endereços a partir de um CEP informado pelo usuário.

O projeto utiliza uma API de CEP para preencher automaticamente os campos de endereço, como rua, bairro, cidade e estado, tornando o formulário mais prático e eficiente.

## Funcionalidades

- Consulta de endereço por CEP
- Preenchimento automático dos campos de endereço
- Campo para CEP
- Campo para rua
- Campo para bairro
- Campo para cidade
- Campo para estado
- Layout responsivo para diferentes tamanhos de tela
- Interface simples, limpa e intuitiva
- Separação entre HTML, CSS e JavaScript

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- API de consulta de CEP
